# WEB API

Built api with flask, written with Python 3.7

### Pre-requisites

`pip install -r requirements.txt`

### Run

`python api.py`

### Interact with API

`localhost:5000`

`localhost:5000/system`

`localhost:5000/disk`

OR

`curl http://127.0.0.1:5000/system`

`curl http://127.0.0.1:5000/disk`
